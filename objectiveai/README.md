# ObjectiveAI Rust SDK

This directory contains the Rust SDK packages for ObjectiveAI.

## Packages

- **[objectiveai](./objectiveai/)** - Core Rust crate with data structures, validation, client-side compilation, and HTTP client
- **[objectiveai-wasm-js](./objectiveai-wasm-js/)** - WebAssembly bindings for browser and Node.js environments
